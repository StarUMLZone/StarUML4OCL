This module provides OCL support for StarUML by integrating StarUML and USE OCL.

GenerateUSEFiles.js
--------------------
This script generates ".use" files from the current uml packages.

GenerateSnapshotSheets.js
-------------------------
This script generates ".xls" files from the current uml packages.

GenerateSoilFiles.js
--------------------
This script generates ".soil" files from the opened ".xls" snapshot work books.

ExecuteUSEOCL.js
-----------------
This script is used for the execution of the use ocl tool.

GenerateObjectDiagrams.js
-------------------------
This script generates object diagrams from the data contained in the SnapshotSheets.

docs/
-----
This directory contains the documentation of the module.

tests/
------
This directory contains tests of increasing complexity for this modules.
Each test is represented as a directory containing both the input and the output 
of the test.
